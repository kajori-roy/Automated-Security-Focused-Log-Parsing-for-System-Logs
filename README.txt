Code Instructions
• It is required that Python version 3.5 or higher is installed on the system.
• To run the code, please open the file named ”content analysis.ipynb”and execute
each of its elements.
• All necessary modules will be installed automatically during the execution.
• It is important to note that all files related to the project should be stored in a
single directory to ensure proper execution of the code.
• IPLoM is written as a module

All the outputs will be in excel file. Only the output for IPLoM will be stored in a different directory.
The final outputs are in Classified_logs.csv and Unique_failures.csv
